Join us in grooming Daria to software perfection

Team Intro

At XBrain, we value the kind of open, direct communication that takes us to good places. We’d like, therefore, to extend you the same courtesy by telling you straight about what we hope for in a new team member:

We’d like it if you were open to new ideas.
We’d like it if you were a self-starter.
We’d like it if you had good work ethic and discipline.
We’d like it if you considered conversation integral to building relationships between colleagues.
We’d like it if you erred on the side of overzealousness.
We’d like it if you respected the varied ideas of others as much as you’d like your own ideas to be respected.
We’d like it if you could sustain a symbiotic relationship with your work.
We’d like it if you could think about what we could contribute to your personal goals, as well as what you could contribute to our culture.
We’d love it if you could join us.

Position


Daria is our answer to the ever-growing demand for accessible machine learning software across various industries. Every member of our team plays a significant role in making this answer as functional and optimal as possible.

Software Engineer/Server

Your role at XBrain would be to develop the server-side of Daria, including RESTful API developments. This role is comprised of two parts—constructing APIs that users can use to easily operate trained models, and internal developers can use for Daria’s application programming

We look for the following qualifications in our candidates:

Industry experience (or experience of equivalence) with Web Framework (Python Django, Ruby on Rails, Java & Scala Play, Java Spring, etc.)
A bachelor’s degree in computer engineering / science, or experience of equivalence


We’d welcome the following qualifications:

Experience with systems and software related to Version Control, Issue Tracking, and Continuous Integration (e.g. Git, JIRA, Jenkins)
Experience with Cloud Architecture (e.g. Amazon AWS, MS Azure, Google GCP)
Varied experience with JavaScript Library
Knowledge of Continuous Build, Integration, Test, and Delivery System

Software Engineer/Data Processing Infra

Your role at XBrain would be to build a distributed data processing system that maximizes efficiency in dataset preparation and modeling processes. These processes, in turn, would help users view various analytics on and combine/transform large volumes of data

We look for these specific qualifications in our candidates:

Industry experience with data distributed processing systems (e.g. Distributed File Systems, Distributed Databases, Distributed Processing Models, Distributed Resource Allocator / Scheduling Systems)
A bachelor’s degree in computer science/engineering, or equivalent experience


We welcome these specific qualifications:

Experience with projects related to Apache Data Processing Systems (HDFS, HBase, Kafka, YARN, Hive, Spark)
Open source experience with other data processing systems
Competence with Java and Python
Experience with systems and software related to Version Control, Issue Tracking, and Continuous Integration (e.g. Git, JIRA, Jenkins)

Machine Learning Engineer

As a machine learning engineer at XBrain, you would be one of the main caretakers of Daria’s automated machine learning technology, helping to develop and enhance its features.

We look for these specific qualifications in our candidates:

a bachelor’s degree or higher in a quantitative discipline such as Computer Science / Engineering, Mathematics, Engineering, Statistics, Physics
more than a year’s experience of using Software, Framework, and Programming Languages such as scikit-learn, Python, TensorFlow, or R


We welcome these qualifications in our candidates:

academic or practical experience with Machine Learning and / or Artificial Intelligence
experience with General Purpose Programing Languages such as Java, C/C++, Python, JavaScript
experience with Version Control Systems such as Git
a Master’s Degree or higher in Computer Science / Engineering, Machine Learning, or Artificial Intelligence
Participation experience in a Kaggle Competition
experience with Deep Learning Library
understanding of large-scale data processing systems

How To Apply
